First, you'll need 'mac ports' installed (which you may already have), and you'll then need to install the git client itself. Mac ports is a command line utility for installing other command line utilities. The git client will give you access to all the commands you need to run.

Install MAC Develeloper Tools (Xcode)

Before you install mac ports, you'll need the mac developer tools. It's a hefty download at 1GB. Get it here: http://developer.apple.com/tools/xcode/

Install MAC Ports

Once you have the dev tool, then get mac ports here: http://www.macports.org/install.php

You'll need one of the ".pkg" installers listed on the page (first paragraph), once you've worked out which version of OSX you're running.

Install Git

When that's all done, you can install the git client using mac ports.

Open a terminal and enter the following:

> sudo port install git-core

Generate SSH Key

You'll need to generate an SSH key that you'll install inside github. This'll identify your machine with Github and let you access it without having to enter a password.

This is what you need to do (in terminal)

ssh-keygen
[enter passphrase]
> ssh-add id_rsa

(May get error here: "can't open a connection to your authentication agent", if so

>   ssh-agent /bin/bash

then

 > ssh-add

You'll see something like this: 'Identity added: /root/.ssh/id_rsa (/root/.ssh/id_rsa'

then

> vim ~/.ssh/id_rsa.pub

Connecting to Github

Add the public key to your Github account, https://github.com/account.

It should be in Account Setting > SSH Keys

Then

> ssh -vT git@github.com

to verify that we can authenticate
