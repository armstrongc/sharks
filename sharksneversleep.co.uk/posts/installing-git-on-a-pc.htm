Install GIT

First things first, install the git client:

http://git-scm.com/download/win

When installing, select command line only

Install Putty

Next, install Putty. This will enable you to set up an SSH key on your machine so that you can access Github.

http://www.chiark.greenend.org.uk/~sgtatham/putty/download.html

Generate SSH Keys

You'll need to generate an SSH key that you'll install inside github. This'll identify your machine with Github and let you access it without having to enter a password.

Run puttygen located in PUTTY folder (might be in the C:\Program Files\Putty folder) C:\Program Files\Putty\puttygen

In PuTTY Key Generator, select 'SSH-2 RSA' from Parameters on bottom and click 'Generate'

Save private key (*.ppk file) and public key (as .txt) for future use and copy the public SSH key.

Connecting to Github

Add the public key to your Github account, https://github.com/account.

It should be in Account Setting > SSH Keys

Double click on your .ppk file that you saved earlier.
